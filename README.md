پروژه مدیریت کتابخانه شخصی با Flask
توضیح پروژه

این پروژه یک سیستم ساده مدیریت کتابخانه شخصی است که با Flask و SQLite ساخته شده است.
کاربران می‌توانند:

ثبت‌نام و ورود به سیستم

مشاهده داشبورد و لیست کتاب‌ها

اضافه کردن، ویرایش و حذف کتاب‌ها

تولید رمز عبور تصادفی

دریافت یک عکس سورپرایز (سگ یا گربه) با کلیک روی دکمه

اطلاعات ورود پیش‌فرض

برای ورود به سیستم می‌توانید از حساب کاربری پیش‌فرض استفاده کنید:

نام کاربری: admin

رمز عبور: 1234

می‌توانید پس از ورود، کتاب‌ها را مشاهده و مدیریت کنید یا حساب جدید بسازید.
استفاده از داشبورد

اضافه کردن کتاب جدید: روی دکمه “اضافه کردن کتاب جدید” کلیک کنید و فرم را پر کنید.

ویرایش یا حذف کتاب: در لیست کتاب‌ها، گزینه مربوطه را انتخاب کنید.

تولید رمز عبور تصادفی: روی دکمه مربوطه کلیک کنید تا یک رمز تصادفی مشاهده شود.

بخش سورپرایز (API)

در صفحه ورود، یک دکمه با عنوان “سورپرایز شو!” وجود دارد.
با کلیک روی این دکمه:

یک تصویر تصادفی سگ یا گربه نمایش داده می‌شود.

این بخش با یک API خارجی کار می‌کند و نیازی به بارگذاری مجدد صفحه ندارد.
